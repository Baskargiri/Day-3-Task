Window

Window is the global object in a browser, and it can also be treated as the root of the document object model. You can access it as window.window is the execution context and global object for that context's in javaScript.





SCREEN

screen is a small information object about physical screen dimensions.



DOCUMENT

Document is the main object of the potentially visible document object model.document contains the DOM initialized by parsing HTML.The Document object(DOM) is your html, aspx, php, or other document that will be loaded into the browser. The document actually gets loaded inside the window object
